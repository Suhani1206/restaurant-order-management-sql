# 🍽️ Restaurant Order Management System (SQL)

This project is a SQL-based database system built for a friend's café to manage customers, menu, and orders.

## 📌 Features
- Create database and tables
- Insert and manage data
- Perform SELECT queries with conditions
- Use JOIN to combine tables
- Update and delete records

## 🛠️ Technologies
- MySQL
- SQL

## 📂 Tables
- customers
- menu
- orders

## 🎯 Purpose
This project simulates a real-world restaurant system.
